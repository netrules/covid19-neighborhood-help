## GETTING STARTED
Requirements: Node, Git.

git clone https://github.com/brandonbrule/covid19-neighborhood-help.git  
npm install

### Development
npm run start

### Deploy
commit and push to master branch  
npm run deploy  
visit https://brandonbrule.github.io/covid19-neighborhood-help/
